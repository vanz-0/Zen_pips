# Educational Vault

## Overview
The Vault is the platform's educational library containing 63+ verified PDF resources covering trading from beginner to elite institutional level.

## Tier Structure
| Tier | Access | Content |
|---|---|---|
| Foundation | Free (all users) | Basic trading concepts, candlestick patterns, risk management fundamentals |
| Intermediate | Free (registered) | SMC basics, market structure, session timing |
| Advanced | VIP Only | Order block mastery, FVG strategies, multi-timeframe analysis |
| Elite | VIP Only | ICT Masterclass, institutional positioning, algorithmic theory |

## Key PDFs in Vault
- `ZenPips_Institutional_Guide.pdf` — The master guide (sent via Brevo email on registration).
- `ZenPips_Tools_Manifesto.pdf` — Platform tools documentation.

## Upload Protocol
- New educational PDFs can be uploaded via admin dashboard or dropped into `zain_brain/raw/` for ZAIN Brain ingestion.
- The ZAIN Manager extracts key concepts and creates/updates concept files automatically.
