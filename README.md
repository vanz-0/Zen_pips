Welcome to the Zen Pips ecosystem repository. This monorepo contains the infrastructure for automated trading signals, community management, and the premium web terminal.

## Structure
- `/zenpips-web`: The Next.js 16 frontend web terminal.
- `/bot`: Telegram bot and webhook logic.
- `/execution`: Core Python backend automation scripts for Telegram and Supabase integration.

See `ROADMAP.md` for project milestones and progress.
